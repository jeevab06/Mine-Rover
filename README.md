# 🛰️ Mine Rover — Hazard Detection & Terrain Mapping System

> **Patent Filed** | Status: Prototype (v1.0)

A semi-autonomous rover designed to improve safety in underground mines by detecting hazardous gases and identifying uneven or unstable floor structures in real time, reducing the need for humans to physically scout high-risk zones.

## 🚨 Problem Statement
Underground mines pose serious risks from toxic/explosive gas buildup (CO, CH₄, H₂S) and unstable, uneven terrain that can cause accidents. Manual inspection puts workers directly in harm's way before hazards are even confirmed.

## 🤖 What It Does
- **Gas Detection** — Continuously monitors air quality and flags hazardous gas concentrations beyond safe thresholds.
- **Terrain Analysis** — Detects uneven, cracked, or unstable mine floor structures using onboard sensors.
- **Remote Monitoring** — Sends live sensor data to an operator station so decisions can be made without entering the danger zone.
- **Alert System** — Triggers warnings when gas levels or terrain instability cross safety limits.
